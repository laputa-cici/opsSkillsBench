# External Skills Workspace

Use this directory for vendored external skills after license verification.

Expected layout:

```text
sources/skills/
├── <skill-name>/
│   ├── SKILL.md
│   ├── LICENSE
│   └── provenance.json
└── README.md
```

Do not add local synthetic skills here. If a distractor is synthetic, place it inside a task and label it clearly in that task's `source.md`.
